         動作する旨を表すことができます。

         あなたは常に当該バージョンの条件に従ってそれを使用し続けることが
         できます。あなたはまた、PHP Groupが発行するそれ以降の任意のバージョンの
         ライセンスの条件に従って、当該対象コードを使用することもできます。
